FromCIEXYZ

bool FromCIEXYZ(double xyz[3], double *col)

Transforms a color value assumed to be in the CIEXYZ (D50) conversion color space into this BTIColorSpace.

This method transforms color values using relative colorimetry, as defined by the International Color Consortium standard. This means that the XYZ argument values taken by this method are represented relative to the D50 white point of the CIEXYZ color space. For example, white will have the XYZ values of D50, which is defined to be X=0.9642, Y=1.000, and Z=0.8249.

Parameters

xyz

A double array containing the CIEXYZ (D50) color values (X = xyz[0], Y = xyz[1], Z = xyz[2]).

col

A double array with length equal to the number of components in this BTIColorSpace.

Return Value

true if successful; otherwise, false.

ToCIEXYZ

bool ToCIEXYZ(double col[], double *xyz)

Transforms a color value assumed to be in this BTIColorSpace into the CIEXYZ (D50) conversion color space.

This method transforms color values using relative colorimetry, as defined by the International Color Consortium standard. This means that the XYZ values returned by this method are represented relative to the D50 white point of the CIEXYZ color space. This representation is useful in a two-step color conversion process in which colors are transformed from an input color space to CIEXYZ and then to an output color space.

Parameters

col

A double array with length of at least the number of components in this BTIColorSpace.

xyz

Pointer to a double array with length 3 (X = xyz[0], Y = xyz[1], Z = xyz[2]).

Return Value

true if successful; otherwise, false.
