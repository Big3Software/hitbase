BTIColorManagementSystem Class

Initialize

static bool Initialize()

Initializes the color management system.

Return Value

true if successful; otherwise, false.

IsInitialized

static bool IsInitialized()

Check whether the color management system is initialized.

Return Value

true if initialized; otherwise, false.

Terminate

static void Terminate()

Terminate the color management system.

Return Value

None.

GetColorManagementSystem

static BTIColorManagementSystem *GetColorManagementSystem()

Return a pointer to the global color management system object.

Return Value

The pointer to the global color management system object; otherwise, NULL.

LinkColorSpace

bool LinkColorSpace(BTIColorSpace* pIColorSpace)

Link a color space to the color management system. Whether the linked color space is supported by default from the library, the existing implementation will be overwritten with the linked one.

Parameters

pIColorSpace

A pointer to the color space object to link to the color management system.

Return Value

true if successful; otherwise, false.

UnlinkColorSpace

bool UnlinkColorSpace(const enum BTColorSpace cs)

Unlinks a color space from the color management system. Whether the unlinked color space is a default color space from the library, then it will be replaced with the libraries default implementation.

Return Value

true if successful; otherwise, false.
