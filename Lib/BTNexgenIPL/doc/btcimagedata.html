Combine

bool BTCImageData::Combine(BTCImageData *pImage, enum BTCombineFunction cf, int x /* = 0 */, int y /* = 0 */, int nOpacity /* = 100 */, bool bClipValues /* = true */, int nDivisor /* = 1 */, int nBias /* = 0 */)

Combines another image with this image at position (x,y). For color depths <= 8 bpp the resulting colors are mapped to the origin color table.

Parameters

pImage

The image to be combined.

cf

Sets the method for combining the images. The color data from the two images is combined on a pixel-by-pixel basis according to the function. The following functions are available:

  • Normal
  • Add
  • Subtract
  • Multiply
  • Divide
  • Difference
  • Lightest
  • Darkest
  • Average
  • Or
  • And
  • Screen
  • Overlay

x

Specifies the logical x-coordinate of the position.

y

Specifies the logical y-coordinate of the position.

nOpacity

At 0% opacity, the combined image is completely transparent; at 100% opacity, the comined image is completely opaque.

bClipValues

Determines how to handle final color values greater than 255 and less than 0.

bClipValues Value < 0 Value > 255
true Value = 0 Value = 255
false Value = 256 + Value Value = Value - 256

nDivisor

The divisor [0..255] is applied to the color values produced by the formula/function. It is used to divide the color values. It can reduce the effects of the other parameters.

nBias

The bias [-255..255] is applied to the color values produced by the formula/function. It shifts each color value by a fixed amount. The number is added to the color value produced by the formula/function and divisor.

Return Value

true if successful; otherwise, false.
